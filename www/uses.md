# Software

Lists are very incomplete, and may not reflect current state.

## General software

* Operating systems:
  * Desktop/laptop: [Arch Linux](https://archlinux.org/), [FreeBSD](https://www.freebsd.org/)
  * Servers: [Debain](https://www.debian.org/), [FreeBSD](https://www.freebsd.org/)
* Desktop environment:
  * Wayland:
    * Compositor: [sway](https://swaywm.org/)
    * Terminal: [foot](https://codeberg.org/dnkl/foot)
    * Statusbar: [waybar](https://github.com/Alexays/Waybar)
    * Notification daemon: [swaync](https://github.com/ErikReider/SwayNotificationCenter)
    * Launcher: [wofi](https://hg.sr.ht/~scoopta/wofi)
    * Screen locker: [swaylock](https://github.com/swaywm/swaylock)
  * X11 (when drivers do not fully support Wayland):
    * Window manager: [i3](https://i3wm.org/)
    * Terminal: [rxvt-unicode](https://software.schmorp.de/pkg/rxvt-unicode.html)
    * Statusbar: [polybar](https://github.com/polybar/polybar)
    * Notification daemon: [dunst](https://dunst-project.org/)
    * Launcher: [rofi](https://github.com/davatorium/rofi)
    * Screen locker: [XScreenSaver](https://www.jwz.org/xscreensaver/)
* Shell: [fish](https://fishshell.com/)
* Editor: [neovim](https://neovim.io/)
* Terminal multiplexer: [tmux](https://github.com/tmux/tmux)
* Browser: [LibreWolf](https://librewolf.net/) or [Firefox](https://www.firefox.com/)
* Music player: [MPD](https://www.musicpd.org/) + [ncmpcpp](https://rybczak.net/ncmpcpp/)
  * [Navidrome](https://www.navidrome.org/) for remote access

## Server software

* Reverse proxy: [SWAG](https://github.com/linuxserver/docker-swag)
* DNS server: [Pi-hole](https://pi-hole.net/)
* Password manager: [Vaultwarden](https://github.com/dani-garcia/vaultwarden)
* Docker management: [Portainer](https://www.portainer.io/)
* Git forge: [Forgejo](https://forgejo.org/)
* Server/home monitoring:
  * Data collection: [telegraf](https://github.com/influxdata/telegraf)
  * Database: [InfluxDB 2](https://www.influxdata.com/)
  * Visualization: [Grafana](https://grafana.com/)
* Fediverse server: [Sharkey](https://activitypub.software/TransFem-org/Sharkey)
* RSS reader: [tt-rss](https://tt-rss.org/)
* Personal website server: [NGINX](https://nginx.org/)
* MQTT server: [mosquitto](https://mosquitto.org/)
